ModelFLOWs-app, an open source Software for data post-processing, patterns identification and development of reduced order models using modal decomposition and deep learning architectures. {: .text-nowrap}

In the **Resources** menu you can find the **Downloads** section where you will be able to download a web-browser version of the application, developed using *Streamlit*, as well as all the python mains for each module. {: .text-nowrap}

The web-browser version serves as a demo and its use is limited. {: .text-nowrap}

In addition to the previously metioned resources, you can find a video tutorial of each module under the **Tutorials** section. These videos explain the mathematics behind each module of the application with an example. {: .text-nowrap}
